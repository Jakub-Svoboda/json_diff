# json_diff
Visual JSON diff tool


## How to run locally

### Use Poetry (see [official docs](https://python-poetry.org/))

#### Instal Poetry

```bash
curl -sSL https://install.python-poetry.org | python3 -
```

#### Start venv

```bash
poetry env use python3.10
poetry shell
```

#### Install dependecies

```bash
poetry install
```
